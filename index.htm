At the core of it all...

Tech Stack

  • Raspbian OS
  • Mostly Python Services
  • Managed by Supervisor
  • Small amount of C/Python wrapper code
  • Connectivity/Power Fault tolerant

Hardware Control

  • Polls NFC sensor for a valid tag
  • Gives visible/audible feedback when found
  • Adds detected tags to work queue (RabbitMQ).
  • Synchronous for payments
  • Asynchronous for everything else

API Client

  • Reads from the work queue
  • Posts tag data to Spigot API via Wifi
  • On error, requeues tag data.
